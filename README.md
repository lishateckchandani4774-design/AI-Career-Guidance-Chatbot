# 🤖 AI Career Guidance Chatbot

An AI-powered Career Guidance Chatbot built using **Python, Streamlit, NLP (TF-IDF), and Scikit-learn**. The application helps users discover suitable career paths, analyze skill gaps, explore learning resources, and receive personalized career recommendations.

---

## 🚀 Features

- 🤖 AI-powered Career FAQ Chatbot
- 🎯 Career Recommendation using NLP (TF-IDF)
- 📊 Match Score Calculation
- 🛠 Skill Gap Analysis
- 📚 Learning Resources
- 🏆 Recommended Certifications
- 📄 Resume Recommendations
- 🗺 Personalized Career Roadmap
- 📈 Career Comparison Dashboard
- 📥 Download Career Report

---

## 🛠 Technologies Used

- Python
- Streamlit
- Pandas
- Scikit-learn
- TF-IDF Vectorization
- Cosine Similarity
- Plotly

## ▶️ Installation

Clone the repository

```bash
git clone https://github.com/lishateckchandani4774-design/AI-Career-Guidance-Chatbot.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
